## 使用python制作神经网络

> 包含三个函数
> - 初始化函数-设定输入层节点,隐藏层节点和输出层节点的数量
> - 训练函数-学习给定训练集样本后,优化权重
> - 查询-给定输入,从输出节点给出答案

### 数据集
